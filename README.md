# Otis - the handwriting tool for Reachy


## Software installation

* From Pypi: ```pip install reachy-otis```
* Or from the source: ```pip install -e otis_folder```

## Fast Kinematic

![](doc/representation.png)

- forward: 40 µs
- inverse: 55 µs
